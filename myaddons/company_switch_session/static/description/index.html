Company Switch Session (Odoo 12)

Improve your multi-company workflow by allowing users to switch their working company per browser tab/session, without changing the default company stored on the user record.

Main Features

  • Per‑tab company selection — each browser tab can work in a different company.
  • No write on user company — the database field res.users.company_id is never updated when switching.
  • Context‑driven queries — ORM, default values and record rules use the company from the current page context.
  • Compatible with standard UI — reuses Odoo's native company selector in the top bar.

How It Works

  • The selected company is stored in the HTTP session and browser storage.
  • The server injects this company into company_id, force_company and allowed_company_ids in the context.
  • Record rules that depend on user.company_id are evaluated using the session-selected company.

Usage

  1. Install the module company_switch_session on Odoo 12.
  2. Log in as a user with access to multiple companies.
  3. Use the company selector in the top-right corner to change company.
  4. Open multiple tabs and select different companies in each tab.
